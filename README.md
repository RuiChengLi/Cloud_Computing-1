## 操作注意

- #### 第一次使用文件夹时:

   * git init
     * git remote add origin remote_repository_URL 
       * git remote -v
         * git pull origin master //拉下来

  - #### 之后上传

     * git status
       * git add .
         * git commit -m “message”
           * git push -u origin master

- #### 如果遇到本地仓库不一致的情况下:
  
  * git pull origin 
    * git merge origin master
      * git push origin 
  
- #### 提醒自己，已经创建过文件夹了..不要再创建了.....
